Mail on Exception.

This plugin will send mails to specified recipients iff there is an exception that an end user can see. But you can always send mails for back-end exceptions.

Usage Comments

You would need to remove Internal Server Error(500) url mapping from the project, as the plugin handles it.

"500"(controller: "report", action: "error")

But you can always create a report folder in views and add error.gsp to the project to override the plugin error page.
